Inventario Moderno V1.0.0

Instrucciones:
1. Copia la carpeta descomprimida a htdocs (XAMPP).
2. Accede a: http://localhost/inventario_moderno_theme/install.php
   - Esto crea la BD 'inventario_moderno' y tablas, además del usuario admin/admin123.
3. Borra install.php después de usarlo por seguridad.
4. Accede a: http://localhost/inventario_moderno_theme/
5. Perfil -> elegir tema (azul, verde, rojo, gris, naranja).
